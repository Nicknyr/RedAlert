Classic stragety games rebuilt for the modern era

Open Source. Free. For all modern operating systems

Which Build?

Two types of build are available, depending on how you want to play:

Stable releases are recommended for players who want to play multiplayer games online.

Playtests provide a preview of the latest improvements for players who want to help test and shape the future releases of OpenRA.

The latest stable release is release-20171014.

Download OpenRa for Windows

The default GPU drivers included with Windows do not support OpenGL rendering. You may need to install full drivers supplied by your GPU vendor.

OpenRA is also available on Chocolatey.

Download OpenRa for OSX

OpenRA requires Mono 4.6 or greater. Download Mono.

OpenRA is also available on Homebrew Cask:

$ brew cask install openra

We do not maintain this external package source, so there may be delays when a new version is released. Please contact the downstream repository maintainers about any packaging issues.

Download OpenRa for Debian

Download and run one of the packages below to install OpenRA via the Software Center.

OpenRA is also available on PlayDeb.

We do not maintain this external package source, so there may be delays when a new version is released. Please contact the downstream repository maintainers about any packaging issues.

Install OpenRA for Fedora / openSUSE

Stable releases and playtests are available via our openSUSE Build Service repository.

Download OpenRa for Arch Linux

Stable releases are available in the official Arch repository, and can be installed using pacman:

$ pacman -S openra

We do not maintain this external package source, so there may be delays when a new version is released. Please contact the downstream repository maintainers about any packaging issues.

Download OpenRa for Gentoo

Unofficial packages for both releases and playtests are available via the “dr” overlay:

$ emerge -av layman

$ layman -a -S dotnet

$ layman -a -S dr -o http://github.com/cerebrum/dr/raw/master/repo.xml

$ printf '%s\n' "games-strategy/openra ~amd64" >> /etc/portage/package.keywords

$ emerge games-strategy/openra

We do not directly maintain this external package source, so there may be delays when a new version is released.Please contact the downstream repository maintainers about any packaging issues.

Install OpenRA for Exherbo

Packages for both releases and playtests are available via the hasufell overlay:

$ cave resolve repository/hasufell

$ cave resolve games-strategy/openra:release

$ cave resolve games-strategy/openra:playtest

We do not directly maintain this external package source, so there may be delays when a new version is released.Please contact the downstream repository maintainers about any packaging issues.

Download OpenRa for FreeBSD

Stable releases are available in the FreeBSD Ports Collection:

# cd /usr/ports/games/openra

# make install
